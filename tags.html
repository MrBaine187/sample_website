Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below doesn't include all the HTML tags, but they do list many of the most importnant ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div Division of section Used in CSS to group block elements
header Creates a header Container for multiple links
ol Creates an ordered list List items are numbered
ul Creates unordered list List items have bullet points

Inline Elements

Tag Name Purpose Example Result
em Emphasize Make Emphasized Text <em>Technical Sophistication</em> Technical Sophistication
strong Strong Make text bold <strong>Technical Sophistication</strong> Technical Sophistication
a Anchor Create a hyper link <a href="#"/> Technical Sophistication
img Image Add Image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span Span adds a hookup to part of a document <span>Technical sophistication</span> Technical Sophistication
li List Item Adds item to list <li>Item 1</li>
